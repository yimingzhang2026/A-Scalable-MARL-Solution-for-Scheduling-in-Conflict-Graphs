# A-Scalable-MARL-Solution-for-Scheduling-in-Conflict-Graphs

This repository contains the initial code implementation for the paper "Distributed MARL for Scheduling in Conflict Graphs" presented at the 2023 59th Annual Allerton Conference on Communication, Control, and Computing.
